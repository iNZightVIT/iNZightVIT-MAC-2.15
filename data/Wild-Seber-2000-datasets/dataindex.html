    • Index for Data-set Files for Chance Encounters

    This file indexes the data-set files for "Chance Encounters" by Wild and Seber. It includes the data sets containing more than 10 points for all chapters (including Chapters 13 and 14 which are available from the web site).

    All files are in tab-separated-text format and have ".txt" extensions. This format is easily read by all major statistical packages and spreadsheets using "Open" under the "File" menu. Even more conveniently, in Windows 95 or 98, if you drag and drop the file icon on to a Minitab, SPSS, or Excel shortcut icon, the data set opens appropriately in the program's spreadsheet.
